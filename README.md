## Prisma Connect with Supabase for NextJS

### Prisma Dependencies Install

#### Dev dependencies:
```terminal
npm install prisma tsx @types/pg --save-dev
```

#### Production dependencies:
```terminal
npm install @prisma/client @prisma/adapter-pg dotenv pg
```
---

### Prisma Initialize
```terminal
npx prisma init 
```
